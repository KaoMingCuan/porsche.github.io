🚗 Porsche Showcase Web Platform

A modern responsive web application that simulates an online showroom for Porsche vehicles, allowing users to explore car models, view specifications, search products, watch promotional videos, and locate charging stations on an interactive map.

📌 Project Overview

This project was developed for the Web Programming course with the goal of building a complete front-end website that combines:

Product showcase

Dynamic search

Multimedia content

Interactive maps

Responsive UI/UX design

The system provides users with an experience similar to a real digital car showroom where they can browse Porsche vehicles and find nearby charging stations easily.
